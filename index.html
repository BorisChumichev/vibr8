Vibr8 is a lightweight (1.2K gzipped) Vibration API wrapper. It reduces inconsistencies of API’s implementations across browsers and gives you some extra powers.

Install

The simplest way to install Vibr8 is using Bower package manager: bower install vibr8. You also can manually download it's source code with examples or minified version. Vibr8 packaged as UMD module to be loaded anywhere, just plop it into a script tag in some html: 

<script src="vibr8.min.js"></script>

A window global named Vibr8 will be created, or if a module system is detected in the host environment, it will be used. You can also reqire Vibr8 using Browserify.

Documentation

Vibr8 is a singleton class, you can instatiate it using the constructor: 

var vibr8 = Vibr8()
